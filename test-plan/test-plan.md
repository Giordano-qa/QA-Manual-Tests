# Test Plan - Login Feature

---

## 1. Objetivo
Validar a funcionalidade de login, garantindo que usuários consigam acessar o sistema de forma segura e consistente.

---

## 2. Escopo
Funcionalidades testadas:
- Login com credenciais válidas
- Validação de senha inválida
- Validação de campos obrigatórios
- Validação de formato de email
- Comportamento do campo senha

---

## 3. Tipos de Teste
- Teste funcional
- Teste de validação
- Teste exploratório

---

## 4. Ambiente de Teste
- Navegador: Google Chrome
- Sistema operacional: Windows 10

---

## 5. Critérios de Entrada
- Sistema disponível
- Usuário previamente cadastrado

---

## 6. Critérios de Saída
- Todos os test cases executados
- Bugs identificados e documentados

---

## 7. Riscos
- Instabilidade da aplicação
- Dados inconsistentes
